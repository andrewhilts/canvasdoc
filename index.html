Unicode PDF Client-side Generation P.O.C.

This outputs a dynamic-length PDF from a HTML source.

Current HTML elements supported are limited to lists and paragraphs.
Styling is ignored, and other block level elements are treated as line breaks and don't have spacing between them, unlike paragrahs.

Right to left is supported, but there are some issues about text flow in browsers that don't support CanvasRenderingContext2D.direction (Only works properly in Safari, or Chrome with experiemental canvas features enabled).

The PDF's page size is customizable.

Source code: https://github.com/andrewhilts/canvasdoc
